## ğŸš€ Getting Started

### Installation
1. Clone or download the repository
2. Navigate to your desired template folder
3. Open `index.html` in your browser

### Customization
Each template can be easily customized:
- **Colors**: Modify CSS variables in `style.css`
- **Content**: Update text and links in `index.html`
- **Images**: Replace placeholder images in `assets/` folder
- **Social Links**: Update href attributes with your actual social media URLs

### QR Code Setup
The QR codes automatically generate based on your page URL. For custom vCard data, modify the JavaScript configuration in `script.js`.
